# Start dev server
以下コマンド実行後に、 http://localhost:1234/index.html でアクセスできます。
```bash
$ npm start
```

# Build & Deploy 
```bash
$ npm run build
# コマンド実行後にデプロイ先のURLなどが出ます。
$ firebase deploy
```
